# Online Food Ordering System (Agile Scrum Project)

## Project Overview
This project is an Online Food Ordering System developed using Agile Scrum methodology and managed through Jira. It simulates real-world Business Analyst and Project Management practices.

## Key Features
- User Authentication (Login/Signup)
- Food Menu & Cart Management
- Order Processing System
- Online Payment Integration
- Delivery Tracking Module

## Agile Methodology
- Jira used for project tracking
- Created Epics, User Stories, and Tasks
- Sprint Planning, Grooming, and Reviews conducted
- Kanban & Scrum boards used for workflow tracking

## Project Documentation
- BRD (Business Requirements Document)
- SRS (Software Requirements Specification)
- Sprint Reports
- Workflow Diagrams

## Tools Used
- Jira
- Agile Scrum
- Kanban Board
- Documentation Tools (Word)
